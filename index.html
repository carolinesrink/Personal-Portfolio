PORTFOLIO


Clicky Game

A full stack React game where users click through memes and try not to click the same one twice. The game will save the high score and restart if the same meme is clicked twice.

Game of Thrones Trivia

A front end 20 question trivia game based on the popular HBO series Game of Thrones.

FriendFinder

A full stack application that has a user answer ten questions and pairs different users together based on final scores.

Bob's Burgers Game

A full stack application that allows a user to input a new burger and devour previously inputed burgers, as well as burgers recently inputed.

Moody App

A front end application that utilizes Spotify, Face++, and Cloudinary APIs and allows users to submit a selfie. Moody will return a Spotify playlist based on the mood analyzed in the photo.

Television GifTastic

A front end application that has already inputed TV shows to choppse from and allows users to search for new TV show gifs by title. The user can then pause and start the gifs returned by the search.

Zig-Rig

A full stack React application that utilizes simple, easy-to-use online planning tool for calculating electrical power distribution for a multitude of power sources.

World Data Vizualization

A full stack application that utilizes D3 technologies that allows a user to select a country, category, and year and receive related New York Times articles.
